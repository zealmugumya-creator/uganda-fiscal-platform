How Each Connection Type Works
🔄 API Sync (QuickBooks/Sage)
TaxLink polls your accounting software every 5 minutes. Any new invoice is automatically pulled, formatted, and submitted to EFRIS. Fully automatic — zero manual steps.
🗂 Desktop Plugin (Tally)
A lightweight plugin installed on the Tally computer. Detects new invoices saved in Tally and submits them automatically. Works even without internet — queues and submits when connection is restored.
📤 Excel Batch Upload
Drop an Excel file with invoices into TaxLink Connect. System validates, maps columns, and submits all invoices to EFRIS in bulk. Returns an Excel with FDN numbers added. Best for field sales teams.
