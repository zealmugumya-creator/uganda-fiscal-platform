How EFRIS Bridge Works
Technical architecture and the user experience for both USSD and WhatsApp paths
Technical Architecture
1
Trader dials *284# on any MTN/Airtel phone
Africa's Talking USSD gateway receives the session
2
Bridge API receives session
Node.js backend validates phone number, fetches business profile
3
USSD menu guides trader
3 inputs: amount, category, buyer TIN (optional)
4
Bridge formats URA EFRIS JSON payload
Builds compliant invoice, adds digital signature, encrypts
5
Submits to URA EFRIS API
efris.ura.go.ug real-time endpoint. Response in <2 seconds
6
FDN returned by SMS to trader
Fiscal Document Number confirms receipt. Total time: 4–6 seconds
